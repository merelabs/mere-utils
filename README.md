# mere-utils-lib

## Description
Utility library contains all the reusable functions for the components that are used in mere projects. 

## Components
- AppUtils
- EnvUtils
- BinUtils
- FileUtils
- FolderUtls
- i18Utils
- IconUtils
- PathUtils
- ProcessUtils
- SignalUtils
- StringUtils
